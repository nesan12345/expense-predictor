# 💸 Personal Expense Predictor

A machine learning project that predicts monthly expenses for hostel students — built as a BYOP capstone for the *Fundamentals of AI and ML* course.

---

## The Problem

As a hostel student, I get a fixed monthly allowance and I almost always end up spending more than I planned — especially around festival months or when college suddenly needs fees or printing. By the time I notice I'm running low, it's already the 20th of the month.

This project tries to solve that by training a regression model to predict total monthly spending based on allowance, rent, and spending patterns across categories like food, recharge, transport, and entertainment.

---

## What It Does

- Generates a realistic synthetic dataset of 300 student-months of spending data
- Trains and compares 5 regression models:
  - Linear Regression
  - Ridge Regression
  - Decision Tree
  - Random Forest
  - Gradient Boosting ← best performer (R² = 0.73, MAE ≈ ₹561)
- Saves 5 output charts: model comparison, actual vs predicted, feature importance, correlation heatmap, and seasonal spending pattern
- Runs entirely from the command line — no GUI needed

## How to Run

```bash
# Clone the repo
git clone https://github.com/<your-username>/expense-predictor.git
cd expense-predictor

# Install dependencies
pip install -r requirements.txt

# Run the full pipeline
python main.py
```

That's it. All outputs will appear in the `outputs/` folder.
